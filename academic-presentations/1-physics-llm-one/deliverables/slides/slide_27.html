Diagonal Probing Confirms Locality

Result 5 — Evidence
  • Diagonal probing: restrict the linear probe to only positions $i \pm 1$ (the token itself and its immediate neighbors)
  • For GPT-2 with relative/rotary PE: accuracy drops only marginally compared to full-attention probing
  • The gap is larger for encoder models (DeBERTa), which spread information more globally
  • Confirms that GPT-2 encodes NT information right at the boundary, not distributed across the sequence
  • This is consistent with a dynamic programming interpretation: each boundary position accumulates parse state from below
Conclusion: GPT-2 concentrates parse information locally at NT boundary positions. The full-attention probe barely outperforms the diagonal one — meaning almost all structural information is available within a window of $\pm 1$ token around each boundary. This locality is a signature of dynamic-programming-style computation.
