Canon Implementation: Just a Few Lines

create_canon() — creating a Canon layer
def create_canon(dim, config): canon = ShortConvolution( hidden_size=dim, kernel_size=config.canon_kernel, # default: 4 activation='silu' if config.canon_activation else None, ) canon._zeyuan_residual = config.canon_residual return canon
apply_canon() — the core forward pass
def apply_canon(store_name, canon, hidden_states, cache, layer_idx, mask): hidden_states2, conv_state = canon(x=hidden_states, mask=conv_mask, cache=conv_state, ...) if canon._zeyuan_residual: return hidden_states + hidden_states2 else: return hidden_states2
ShortConvolution wraps nn.Conv1d with causal padding and optional causal_conv1d CUDA kernel for speed.
Key line: The residual connection h + conv1d(h) is what makes Canon work — preserving vertical gradient flow while adding horizontal mixing.
